    • INTRODUCCION BÁSICA AL LENGUAJE HTML

    El lenguaje HTML (HyperText Markup Languaje) es un lenguaje de marcas (etiquetas) de hipertexto. Es el lenguaje con el que se diseñan las páginas web. Estas páginas constituyen una forma eficaz de comunicación capaz de llegar a millones de personas.

      ¿Que es el html y para que lo usamos?
    • HTML significa lenguaje de marcado de hipertexto
    • HTML es el lenguaje de marcado estándar para crear páginas web.
    • HTML describe la estructura de una página Web
    • HTML consta de una serie de elementos
    • Los elementos HTML le dicen al navegador cómo mostrar el contenido
    • Los elementos HTML etiquetan piezas de contenido como "este es un encabezado", "este es un párrafo", "este es un enlace", etc.

      ¿Cuáles son las líneas de códigos que tengo que escribir para crear mi archivo index.html?
    • Para crear un archivo index, hay que escribir algunas lineas fundamentales de codigo, las cuales van a definir el esqueleto de mi pagina.
    • La primera linea que debemos de escribir, es < html> para iniciar un documento, y < /html> al final de la pagina para cerrarlo, todo ello sin los espacios.
    • Despues pondremos < head> < /head> , entre estas dos etiquetas van a ir todo los links a paginas de librerias tipo bootstrap
    • A continuación viene < body> < /body> , dentro del body va a tener lugar todo el contenido de nuestro código.
    • A aqui se puede ver como quedaría la estructura principal del archivo index.html.
