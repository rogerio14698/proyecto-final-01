    • ELEMENTOS DEL LENGUAJE HTML

    El elemento HTML es todo, desde la etiqueta de inicio hasta la etiqueta final:

    < nombre de la etiqueta> El contenido < /el nombre de la etiqueda>

      Algunos ejemplos de etiquedas mas usadas
    • < h1> con su cierre, "se usa para titulos" o letras mas grandes que el párrafo< / h1>
    • < p> Para escribir párrafos, con su cierre < /p>
      Importante no olvidar el cierre de las etiquetas.

      Ya que el codigo no va a funcionar y va a salir error:

    • < h1 si no pongo el cierre, el titulo no funciona

    • < h1 pero si pongo el cierre, si funciona < / h1>
